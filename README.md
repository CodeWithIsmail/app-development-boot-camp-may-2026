## 🛠️ Local Setup
This project uses Firebase for backend services. For security, configuration files (`google-services.json` and `firebase_options.dart`) are excluded.

### Steps to Run:
1. **Create a Firebase Project** at [Firebase Console](https://console.firebase.google.com/).
2. **Setup FlutterFire**: Run `flutterfire configure` in your terminal. This will automatically generate the missing files.
3. **Run the App**: `flutter run`
